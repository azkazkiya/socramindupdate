Konsep Struktur Kontrol Percabangan

Apakah kalian senang bermain game? Kapan kalian bisa naik ke level berikutnya? Bagaimana jika kalian terkena serangan musuh sebelum maju ke level berikutnya? Lalu bagaimana jika kalian bisa menyerang semua musuh? Tentu jika berhasil, kalian akan berproses menuju level selanjutnya bukan?

Demikian pula ketika kalian bermain bola voli. Apa yang terjadi jika salah satu tim gagal menyelamatkan bola? Pastinya tim lawan akan mendapatkan poin dan melakukan perayaan, kemudian permainan akan dilanjutkan. Contoh tersebut merupakan kondisi yang berkaitan dengan kejadian bersyarat atau percabangan, yang ditandai dengan kata ‘jika’ dan ‘maka’.

1. Pemilihan dengan Satu Kondisi

Struktur pemilihan dengan satu kondisi, atau sering disebut sebagai pernyataan if, adalah bentuk paling dasar dari pengambilan keputusan dalam pemrograman. Struktur ini digunakan untuk menjalankan satu blok kode hanya jika sebuah kondisi tertentu terpenuhi (bernilai benar atau true). Jika kondisi tersebut tidak terpenuhi (bernilai salah atau false), maka blok kode di dalamnya akan dilewati dan program akan melanjutkan eksekusi ke baris kode berikutnya setelah blok if. 


nilai = 85

# Jika nilai lebih besar dari 75, maka cetak pesan "Selamat, Anda lulus!"
if nilai > 75:
    print("Selamat, Anda lulus!")

print("Pengecekan nilai selesai.")
2. Pemilihan dengan Dua Kondisi

Struktur if-else digunakan ketika ada dua kemungkinan aksi yang harus dipilih berdasarkan sebuah kondisi. Jika kondisi pada if terpenuhi (true), maka blok kode pertama akan dieksekusi. Namun, jika kondisi tersebut tidak terpenuhi (false), maka program akan secara otomatis menjalankan blok kode yang ada di dalam else. Dengan kata lain, ini adalah situasi "jika kondisi A benar, lakukan ini; jika tidak, lakukan itu". 


umur = 15

# Memeriksa apakah umur cukup untuk mendapatkan SIM
if umur >= 17:
    print("Anda sudah cukup umur untuk membuat SIM.")
else:
    print("Maaf, Anda belum cukup umur untuk membuat SIM.")
3. Pemilihan dengan Lebih dari Dua Kondisi

Ketika terdapat lebih dari dua kemungkinan jalur eksekusi, kita bisa menggunakan struktur if-elif-else (di Python) atau struktur switch-case (di bahasa lain seperti C/C++ dan Java).

a) Struktur if-elif-else

Struktur ini memungkinkan kita untuk mengevaluasi serangkaian kondisi secara berurutan. Program akan memeriksa kondisi pertama (if). Jika benar, blok kodenya dieksekusi dan sisa struktur dilewati. Jika salah, program akan lanjut memeriksa kondisi kedua (elif), dan begitu seterusnya. Jika tidak ada satupun kondisi yang terpenuhi, maka blok else di bagian paling akhir yang akan dieksekusi sebagai pilihan default. 


nilai = 78

# Mengkategorikan nilai menjadi A, B, C, D, atau E
if nilai >= 85:
    print("Nilai Anda: A")
elif nilai >= 75:
    print("Nilai Anda: B")
elif nilai >= 60:
    print("Nilai Anda: C")
elif nilai >= 50:
    print("Nilai Anda: D")
else:
    print("Nilai Anda: E")
b) Struktur Switch-Case

Di bahasa pemrograman lain seperti C, struktur switch-case adalah alternatif dari if-elif-else yang lebih efisien ketika kondisi yang diperiksa adalah perbandingan kesamaan (==) terhadap sebuah variabel tunggal. Pernyataan break sangat penting untuk menghentikan eksekusi agar tidak "jatuh" ke case berikutnya. 


#include 

int main() {
    char pilihan = 'B';

    // Menentukan aksi berdasarkan pilihan menu
    switch (pilihan) {
        case 'A':
            printf("Anda memilih untuk menambah data.\n");
            break;
        case 'B':
            printf("Anda memilih untuk mengubah data.\n");
            break;
        case 'C':
            printf("Anda memilih untuk menghapus data.\n");
            break;
        default:
            printf("Pilihan tidak valid.\n");
            break;
    }

    return 0;
}
